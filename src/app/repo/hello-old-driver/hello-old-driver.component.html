REGEXP analyser

Remove unnecessary page contents, and list "secret codes" only.

Hidden contents are also visible as long as they are in the HTML source code.

Auto-detect domain name

Search the true domain name, no matter how it changes.

LLSS changes domain name frequently. Thanks to the publisher page, it is able to retrieve the latest domain name.

Multi-threading

Crawl pages according to Wordpress RSS Feed.

Unlink traditional crawlers, it reads from RSS Feed first to get latest posts. Instead of scanning thousands of pages, it is usually a better choice to read from new posts only.

Usage

llss [-h] [-p protocol] [-d domain] [page]
page
The page number of result. Must be greater than 0. Default value is 1.
-d string
Domain used to get the feed (auto checks domain if not given)
-h
Show help
-p string
Protocol used to get the feed, like "http" or "https" (default "https")
